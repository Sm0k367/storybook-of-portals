# 📖 The Storybook of Portals

A cinematic, interactive web experience that guides users through 10 digital portals — each a unique AI-powered destination built by EpicTech AI.

## 🚀 Features

- 📚 Flipbook-style page layout
- 🎵 Background music
- 💬 Narrator-style storytelling
- 🔗 10 interactive "portal" links
- ⚙️ Pure HTML, CSS, JS — no frameworks

## 🛠️ Setup

1. Clone or download the repo
2. Open `index.html` in your browser
3. Optionally host it via GitHub Pages, Vercel, or Netlify
